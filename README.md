# HRIS Frogs Indonesia

Human Resource Information System built with Laravel and Tailwind CSS.

## Description
This project is an internal HR management system for Frogs Indonesia to manage employee data, OKR, tickets, and internal documents.

## Features
- Authentication with email verification
- Role-based access control (8 roles)
- Employee management (CRUD)
- OKR (goal setting) system per division
- Ticketing system for requests
- Document management (SOP, Form, Memo)
- Role-based sidebar navigation

## Tech Stack
- Laravel
- Tailwind CSS
- MySQL
- Laravel Auth
